## 問題

以下の要件を満たす Car クラスを作成しなさい。<br>
クラスを用いて、車が動いたことを報告してください。

要件
- Car クラスは、modelの情報を持つ。これは```＿init__```で引数を元に作成される。
- Car クラスは moveメソッドを持ち、このメソッドは車が動いたことを以下のよう形で出力する。<br>
出力: "{model}が前進"

inputは以下のようにする。
```python
model= input()
```
## テストケース

[
	{
		"inputs": ["Toyota"],
	},
	{
		"inputs": ["Honda"],
	}
]

## 模範回答
class Car:
    def __init__(self, model):
        self.model = model
        
    def move(self):
        print(f"{self.model}車が前進")

model= input()

car = Car(model)
car.move()

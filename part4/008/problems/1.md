## 問題

再帰関数を用いて、フィボナッチ数列を作る関数を作成しなさい。<br>
ヒント：nが1以下の場合は1を返し、それ以外の場合は前の２つの数の和を返すようにしてみましょう！

inputの方法は以下のようにしましょう！
```python
n = int(input())
```
## テストケース

[
	{
		"inputs": ["3"]
	},
	{
		"inputs": ["6"]
	}
]

## 模範回答
def fibonacci(n):
    if n <= 1:
        return 1
    else:
        return fibonacci(n - 2) + fibonacci(n - 1)

n = int(input())
result = fibonacci(n)
print(result)

## 問題

再帰関数を用いて、数の各桁の合計を計算する関数を作成しなさい。<br>
ヒント：数が111ならば1+1+1で3を返せればよい。123は100+10+1で作られていて、各桁ごとに10倍されているよ！

inputは以下のようにする。
```python
num = int(input())
```
## テストケース

[
	{
		"inputs": ["123"]
	},
	{
		"inputs": ["9876"]
	}
]

## 模範回答
def digit_sum(num):
    if num < 10:
        return num
    else:
        return num % 10 + digit_sum(num // 10)

num = int(input())
result = digit_sum(num)
print(result)

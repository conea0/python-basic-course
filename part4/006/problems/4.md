## 問題

2つの文字列を引数に取り、それらを結合して返す関数を作成しなさい。<br>
引数には"monster","python"をデフォルト値として設定しましょう。

## テストケース

[
	{
		"inputs": ["Hello", "World"]
	},
	{
		"inputs": ["Python"]
	}
]


## 模範回答
def concat_strings(s1="monster", s2="python"):
    return s1 + s2

s1= input()
s2= input()
cs= concat_strings(s1,s2)
print(cs)

## 問題

引数として渡された整数3つを比較し、最大値を返す関数を作成しなさい。<br>
引数には1,2,3をデフォルト値として設定しましょう。

## テストケース

[
	{
		"inputs": ["4", "8", "2"]
	},
	{
		"inputs": []
	}
]


## 模範回答
def max_of_three(a=1, b=2, c=3):
    return max(a, b, c)

a=int(input())
b=int(input())
c=int(input())
mot= max_of_three(a,b,c)
print(mot)

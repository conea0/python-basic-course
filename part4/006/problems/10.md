## 問題

引数として渡された配列の要素をすべて2倍にして返す関数を作成しなさい。<br>
引数には3,40,64をデフォルト値として設定しましょう。

## テストケース

[
	{
		"inputs": ["1 4 7"]
	},
	{
		"inputs": ["-3 6 9"]
	}
]


## 模範回答
def double_elements(arr):
    result = []
    for x in arr:
        result.append(x*2)
    return result

arr = list(map(int, input().split()))
de= double_elements(arr)
print(de)

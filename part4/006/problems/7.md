## 問題

引数として渡された2つの数の差を絶対値で計算して返す関数を作成しなさい。<br>
引数には1,2をデフォルト値として設定しましょう。

## テストケース

[
	{
		"inputs": ["3"]
	},
	{
		"inputs": ["-5", "10"]
	}
]

## 模範回答
def absolute_difference(a=1, b=2):
    return abs(a - b)

a= int(input())
b= int(input())
ad= absolute_difference(a,b)
print(ad)

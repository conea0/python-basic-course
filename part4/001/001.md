### 関数
関数は、プログラム内で再利用可能な処理をまとめたものです。<br>「**関数は、特定の作業をまとめたもので、同じ作業を何度も繰り返すときに便利です**」と言えます。

例えば、お菓子の袋からあめを取り出す作業を考えてみましょう。もし同じ作業を何度も繰り返す必要がある場合、関数を使うことで簡単に処理を実行できます。

実際に上記の例をコードにしてみましょう！
```python
def take_candy():
    print("あめを取り出しました！")
```
このようにあめを取り出したことを報告する作業が関数化できました！

関数は作っただけでは何もできません。<br>
関数は**呼び出してこそ**使えるものなのです！

では次に関数を呼び出して使ってみます！<br>
関数を使って、お菓子を3回取り出す場合、以下のように書くことができます
```python
for _ in range(3):
    take_candy()
```
forループを使って3回の繰り返しを行っています。<br>take_candy()という関数を呼び出すことで、お菓子を3回取り出しを報告する作業がされます。

### 関数化
これまでに習ってきたif文とfor文も関数化してみましょう！

今回は与えられたリストを偶数にする関数を作ってみます。
```python
def print_even_numbers(numbers):
    for num in numbers:
        if num % 2 == 0:
            print(num)
```
解説すると、numbersというリストからfor文で数ひとつづつ取り出してきます。<br>
その数ひとつひとつが２で割った時にあまりが０であるかを確認し、０であったら表示する関数です。

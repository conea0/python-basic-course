## 問題

文字列が回文（前から読んでも後ろから読んでも同じ）かどうかを判定する関数を作成しなさい。
ヒント: 文字列==文字列[::-1]になれば回文であるね

inputの方法
文字列のinputはabcbaのように行われる。以下のコードを利用して良い
```python
string = input()
```

## 模範回答
def is_palindrome(s):
    return s == s[::-1]

string = input()
result = is_palindrome(string)
print(result)

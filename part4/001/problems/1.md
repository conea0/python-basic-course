## 問題

リスト内の数の合計を表示する関数を作成しなさい。

inputの方法
リストのinputは1 2 3 4という形式で行われる。
これをPythonのリストにするために、以下のコードを利用してよい。

```python
numbers = list(map(int, input().split()))
```


## テストケース

[
	{
		"inputs": 
		[
			"1 2 3 4 5",
		],
	},
	{
		"inputs": 
		[
			"10 20 30 40 50",
		],
	}
]


## 模範回答
def sum_numbers(numbers):
    return sum(numbers)

numbers = list(map(int, input().split()))
sum_numbers(numbers)

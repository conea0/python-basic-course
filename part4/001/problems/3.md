## 問題

数値のリストから最大値を求める関数を作成しなさい。

inputの方法
リストのinputは1 2 3 4という形式で行われる。
これをPythonのリストにするために、以下のコードを利用してよい。
```python
numbers = list(map(int, input().split()))
```


## テストケース

[
	{
		"inputs": 
		[
			"1 5 2 8 3",
		],
	},
	{
		"inputs": 
		[
			"10 25 8 16 4",
		],
	}
]


## 模範回答
def find_max(numbers):
    return max(numbers)

numbers = list(map(int, input().split()))
result = find_max(numbers)
print(result)

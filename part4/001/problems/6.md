## 問題

文字列に含まれる単語の数を数える関数を作成しなさい。

inputの方法
文字列のinputはHello World!のように行われる。
```python
sentence = input()
```

## テストケース

[
	{
		"inputs": 
		[
			"Python is awesome!",
		],
	},
	{
		"inputs": 
		[
			"This is a test.",
		],
	}
]


## 模範回答
def count_words(sentence):
    words = sentence.split()
    return len(words)

sentence = input()
result = count_words(sentence)
print(result)

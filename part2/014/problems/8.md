## 問題

リスト s=["Hello", "world", ".", "I", "am", "fine", "!"] の5番目の要素をhappyに変えて出力してください。

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["['Hello', 'world', '.', 'I', 'am', 'happy', '!']"]
  	},
]
```

## 模範回答
```python
s = ["Hello", "world", ".", "I", "am", "fine", "!"]
s[1] = "happy"
print(s)
```

## 問題

リスト s=["北海道", "東北", "関東", "中部", "近畿"] の"北海道"を"九州"に変えて出力してください。

list(map(str, input().split()))を使ってリストを入力しよう。
入力の時は、1つの要素おきにスペースを入れよう。
## テストケース

```json
[
	{
		"input": ["九州　東北　関東　中部　近畿"],
		"output": ["['九州', '東北', '関東', '中部', '近畿']"]
  	},
]
```

## 模範回答
```python
s = list(map(str, input().split()))
s[0] = "九州"
print(s)
```

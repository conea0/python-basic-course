## 問題

リスト s=["a", "b", "c", "d", "e"] の1番目から4番目の要素を2個刻みに取り出して出力してください。

list(map(str, input().split()))を使ってリストを入力しよう。
入力の時は、1つの要素おきにスペースを入れよう。
## テストケース

```json
[
	{
		"input": ["a　b　c　d　e"],
		"output": ["['b', 'd']"]
  	},
]
```

## 模範回答
```python
s = list(map(str, input().split()))
print(s[1:5:2])
```

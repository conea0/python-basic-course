## 問題

リスト s=["January", "February", "March", "April", "May", "June"] の3番目から5番目の要素を取り出して出力してください。

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["['April', 'May', 'June']"]
  	},
]
```

## 模範回答
```python
s = ["January", "February", "March", "April", "May", "June"]
print(s[3:6])
```

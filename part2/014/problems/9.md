## 問題

リスト s=["佐藤", "岩渕", "田中", "高橋", "千葉"] の"田中"を"菊池"に変えて出力してください。

list(map(str, input().split()))を使ってリストを入力しよう。
入力の時は、1つの要素おきにスペースを入れよう。
## テストケース

```json
[
	{
		"input": ["佐藤　岩渕　田中　高橋　千葉"],
		"output": ["['佐藤', '岩渕', '菊池', '高橋', '千葉']"]
  	},
]
```

## 模範回答
```python
s = list(map(str, input().split()))
s[2] = "菊池"
print(s)
```

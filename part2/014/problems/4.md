## 問題

リスト s=["1", "2", "3", "4"] の1番目の要素をhelloに変えて出力してください。

list(map(str, input().split()))を使ってリストを入力しよう。
入力の時は、1つの要素おきにスペースを入れよう。
## テストケース
input = none
```json
[
	{
		"input": ["1　2　3　4"],
		"output": ["['1', 'hello', '2', '3']"]
  	},
]
```

## 模範回答
```python
s = list(map(str, input().split()))
s[1] = "hello"
print(s)
```

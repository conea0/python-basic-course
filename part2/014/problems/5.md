## 問題

リスト s=["a", "b", "c", "d", "e"] の3番目の要素をgoodに変えて出力してください。

list(map(str, input().split()))を使ってリストを入力しよう。
入力の時は、1つの要素おきにスペースを入れよう。
## テストケース

```json
[
	{
		"input": ["a　b　c　d　e"],
		"output": ["['a', 'b', 'c', 'good', 'e']"]
  	},
]
```

## 模範回答
```python
s = list(map(str, input().split()))
s[3] = "good"
print(s)
```

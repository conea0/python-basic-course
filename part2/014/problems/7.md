## 問題

リスト s=["Japaneze", "Math", "Science", "English"] の0番目の要素をMusicに変えて出力してください。

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["['Music', 'Math', 'Science', 'English']"]
  	},
]
```

## 模範回答
```python
s = ["Japaneze", "Math", "Science", "English"]
s[0] = "Music"
print(s)
```

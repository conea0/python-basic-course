## 問題

リスト s=["い", "ろ", "は", "に", "ほ"] の3番目から4番目の要素を取り出して出力してください。

list(map(str, input().split()))を使ってリストを入力しよう。
入力の時は、1つの要素おきにスペースを入れよう。
## テストケース

```json
[
	{
		"input": ["い　ろ　は　に　ほ"],
		"output": ["['に', 'ほ']"]
  	},
]
```

## 模範回答
```python
s = list(map(str, input().split()))
print(s[3:5])
```

## 問題

copyメソッドを使って、list(map(int, input().split()))で入力したリストをコピーし、コピーしたものに10を追加してください。

コピーする前とコピーし追加した後を出力してください。

## テストケース

```json
[
	{
		"input": ["1 2 3"],
		"output": ["['1', '2', '3'],['1','2','3','10']"]
  	},
]
```

## 模範回答
```python
l = list(map(int, input().split()))
l2 = l.copy()
l2.append(10)
print(l)
print(l2)
```

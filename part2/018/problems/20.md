## 問題

list(map(str, input().split()))で入力したリストに、リストl2=['10%引き', '50円引き', '半額']を追加して出力してください。

## テストケース

```json
[
	{
		"input": ["タイムセール　フェア　イベント"],
		"output": ["['タイムセール', 'フェア', 'イベント', '10%引き', '50円引き', '半額']"]
  	}
]
```

## 模範回答
```python
l = list(map(str, input().split()))
l2 = ['10%引き', '50円引き', '半額']
l.extend(l2)
print(l)
```

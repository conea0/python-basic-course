## 問題

list(map(str, input().split()))で入力したリストに、リストl2=['歩く', '走る', '飛ぶ']を追加して出力してください。

## テストケース
input = none
```json
[
	{
		"input": ["歌う　踊る　描く"],
		"output": ["['歌う', '踊る', '描く', '歩く', '走る', '飛ぶ']"]
  	}
]
```

## 模範回答
```python
l = list(map(str, input().split()))
l2 = ['歩く', '走る', '飛ぶ']
l.extend(l2)
print(l)
```

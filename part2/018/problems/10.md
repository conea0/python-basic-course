## 問題

list(map(str, input().split()))で入力したリストのWedを削除して出力してください。

入力は「Sun Mon Tue Wed Thu」としてください。
## テストケース

```json
[
	{
		"input": ["Sun Mon Tue Thu"],
		"output": ["['Sun', 'Mon', 'Tue', 'Thu']"]
  	},
]
```

## 模範回答
```python
l = list(map(str, input().split()))
l.remove('Wed')
print(l)
```

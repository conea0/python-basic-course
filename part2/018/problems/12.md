## 問題

list(map(str, input().split()))で入力したリストをコピーし、waterを追加してください。

コピーする前とコピーし追加したリストを出力してください。
## テストケース

```json
[
	{
		"input": ["melon　cabbage　tomato　potato　onion"],
		"output": ["['melon', 'cabbage', 'tomato', 'potato', 'onion', 'water']"]
  	},
]
```

## 模範回答
```python
l = list(map(str, input().split()))
l2 = l.copy()
l2.append('water')
print(l)
print(l2)
```

## 問題

list(map(str, input().split()))で入力したリストに、リスト['apple', 'orange', 'grape', 'peach']を追加して出力してください。

## テストケース

```json
[
	{
		"input": ["banana　lemon"],
		"output": ["['banana', 'lemon', 'apple', 'orange', 'grape', 'peach']"]
  	}
]
```

## 模範回答
```python
l = list(map(str, input().split()))
l2 = ['apple', 'orange', 'grape', 'peach']
l.extend(l2)
print(l)
```

## 問題

リストl=['banana', 'lemon']に、リストl2=['apple', 'orange', 'grape', 'peach']を追加して出力してください。

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["['banana', 'lemon', 'apple', 'orange', 'grape', 'peach']"]
  	}
]
```

## 模範回答
```python
l = ['banana', 'lemon']
l2 = ['apple', 'orange', 'grape', 'peach']
l.extend(l2)
print(l)
```

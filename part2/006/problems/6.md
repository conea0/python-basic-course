## 問題

入力された文字列を逆順にして3回繰り返して出力してください。(逆順にする方法がわからない場合は調べよう！自分で調べるクセをつけましょう。プログラミングでは動けば良いので、習っていないことでも進んで使いましょう。)

### 例

input:
```sh
hello
```

output:

```sh
olleholleholleh
```

### inputの使い方

文字列を入力として受け取るには、input()を使います。

```python
s = input()
```

## テストケース

```json
[
	{
		"input": ["apple"],
	},
	{
		"input": ["ringo"],
	}
]
```

## 模範回答

```python
s = input()

print((s[::-1]) * 3)
```

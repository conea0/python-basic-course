## 問題

文字列を二つ受け取り、一つ目の文字列の3番目と二つ目の文字列の3番目を結合して出力するプログラムを書いてください。
### 例

input:
```sh
example
computer
```
output:

```sh
am
```

### inputの使い方
文字列を入力として受け取るには、input()を使います。

```python
s = input()
t = input()
```

## テストケース

```json
[
  {
    "input": ["computer", "science"],
    "output": "mi"
  }
]
```

## 模範回答

```python
s = input()
t = input()
print(s[2] + t[2])
```

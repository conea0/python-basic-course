## 問題

4回printを使って、「ABCD」からA, B, C, Dを一文字づつ改行させて出力しなさい。

## テストケース

input = none

```json
[
  {
    "input": [],
    "output": [
      "A",
      "B",
      "C",
      "D"
      ]
  }
]
```

## 模範回答

```python
s = "ABCD"
print(s[0])
print(s[1])
print(s[2])
print(s[3])#「for文」を学ぶと、もっと簡単に書けます。
```

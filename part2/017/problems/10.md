## 問題

変数に「satou　takahashi　tanaka　asaki」をlist(map(str, input().split()))で入力し、逆順に並び変えて表示させてください。

## テストケース

```json
[
	{
		"input": ["satou　takahashi　tanaka　asaki"],
		"output": ["['asaki','tanaka','tahashi','satou']"]
  	}
]
```

## 模範回答
```python
a = list(map(str, input().split()))
a.reverse()
print(a)

```

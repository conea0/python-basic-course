## 問題

list(map(int, input().split()))でリストを入力しその中で、「738」が何番目にあるか出力してください。

ただし、「738」は必ず存在することとします。

## テストケース

```json
[
  {
    "input": ["229 854 523 547 730 923 738 240 992 296"],
    "output": ["6"]
  },
]
```

## 模範回答

```python
l = list(map(int, input().split()))
print(l.index(738))
```

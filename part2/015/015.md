# リストの要素情報取得

## リストでの index

ある要素が、あるリストの何要素目にあるのかを調べたいときは、

「index」というものを使います

---

リスト.index("調べたい要素")

---

例えば以下のようになります

【例題１】

```python
lis = [0, 2, 4, 6, 8]
lis.index(6)
#3
```

上の例のように、複数マッチする場合は、最初のインデックスが返されます。

また、検索対象が存在しない場合はエラーになるので注意してください。

## いくつ存在するか数える

「count」を使うことで、ある文字列がいくつ存在するか調べることができます。

【例題２】

```python
lis = ['a', 'a', 'a', 'a', 'b', 'c', 'c']

print(lis.count('a'))
# 4

print(lis.count('b'))
# 1

print(lis.count('c'))
# 2

print(lis.count('d'))
# 0
#2
```

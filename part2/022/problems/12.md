## 問題

`setdefault` を使用して`numbers_sd = {3 : "number_1",4 : "number_2",1 : "number_3"}`の中に`2`の第一引数だけ追加した後、`get`を使用して`2`の第二引数を出力してください

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["None"]
  	},
]
```

## 模範回答
```python
nombers_sd = {2 : "number_1",3 : "number_2",1 : "number_3"}
numbers_sd.setdefault(2)
print(numbers_sd.get(2))
```

## 問題

`pop` を使用して`words = {"A" : "ち","J" : "ま","Y" : "ん"}`の中から好きなものを削除して、その削除したものを出力してください。

wordsを作成し、それを使って出力しよう。そのままprintは禁止です。

## テストケース

```json
[
	{
		"input": [],
		"output": ["ま"]
  	},
]
```

## 模範回答
```python
words = {"A" : "ち","J" : "ま","Y" : "ん"}
poped_words = words.pop("J")

print(poped_words)
```

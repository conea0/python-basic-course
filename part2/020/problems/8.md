## 問題

A, BがCであるかを出力しなさい。

真であればTrue、偽であればFalseを出力しなさい。

print(True),print(False)は禁止です。変数を使って代入し比較しましょう。

【A,B,Cの組み合わせ】

A:[2,2,2]

B:[2,2,2]

C:等しくない

## テストケース

```json
[
  {
    "input":,
    "output": [False]
  }
]
```

## 模範回答

```python
lisA = [2,2,2]
lisB = [2,2,2]
print(lisA != lisB)
```

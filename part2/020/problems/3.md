## 問題

A, BがCであるかを出力しなさい。

真であればTrue、偽であればFalseを出力しなさい。

print(True),print(False)は禁止です。変数を使って代入し比較しましょう。

【A,B,Cの組み合わせ】

A:[100, -100, 0]

B:[100, -100, 0]

C:等しい

## テストケース


```json
[
  {
    "input":,
    "output": [True]
  }
]
```

## 模範回答

```python
lisA = [100, -100, 0]
lisB = [100, -100, 0]
print(lisA == lisB)
```

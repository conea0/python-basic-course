## 問題

AがB未満であるかを出力しなさい。

真であればTrue、偽であればFalseを出力しなさい。

print(True),print(False)は禁止です。変数を使って代入し比較しましょう。

【A,Bの組み合わせ】

A:[5,1,2]

B:[5,1,2]

## テストケース


```json
[
  {
    "input":[],
    "output": ["False"]
  },
]
```

## 模範回答

```python
lisA = [5,1,2]
lisB = [5,1,2]
print(lisA < lisB)
```

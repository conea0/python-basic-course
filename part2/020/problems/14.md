## 問題

AがB以上であるかを出力しなさい。

真であればTrue、偽であればFalseを出力しなさい。

print(True),print(False)は禁止です。変数を使って代入し比較しましょう。

【A,Bの組み合わせ】

A:[0,-1,1]

B:[0,-1,1]

## テストケース


```json
[
  {
    "input":,
    "output": [True]
  }
]
```

## 模範回答

```python
lisA = [0,-1,1]
lisB = [0,-1,1]
print(lisA >= lisB)
```

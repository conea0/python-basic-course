## 問題

AがB以上であるかを出力しなさい。

真であればTrue、偽であればFalseを出力しなさい。

print(True),print(False)は禁止です。変数を使って代入し比較しましょう。

【A,Bの組み合わせ】

A:[1,2,3,4,5]

B:[1,2,3,4,5]

## テストケース

```json
[
  {
    "input":,
    "output": [True]
  }
]
```

## 模範回答

```python
lisA = [1,2,3,4,5]
lisB = [1,2,3,4,5]
print(lisA >= lisB)
```

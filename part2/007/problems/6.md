# 7

第三引数に`最大置換回数`を指定できます。
「文字列.replace("置換したい文字列", "置換する文字列", "最大置換回数")」
```python
str = "hello world"
print(str.replace("l", "123", 2))
```
上のコードを実行すると、　he123123o world　と表示されます。
さっそく以下の問題に挑戦してみましょう!!

---
## 問題

"one-two-three-four-five-" の-を2個だけ!にして出力してください。

"one-two-three-four-five-"はinput()で入力してください。
## テストケース

```json
[
	{
		"input": ["---"],
		"output": ["!!-"]
  	},
]
```

## 模範回答
```python
s = input()
print(s.replace("-", "!", 2))
```

## 問題

a = {'apple':'りんご','banana':'バナナ','orange':'みかん'}という辞書に、「peach-もも」を追加して表示させてください。

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["{'apple':'りんご','banana':'バナナ','orange':'みかん','peach':'もも'}"]
  	},
]
```

## 模範回答
```python
a = {'apple':'りんご','banana':'バナナ','orange':'みかん'}
a['peach']='もも'
print(a)
```

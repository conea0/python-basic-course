## 問題

二重ループを使って、1から9の掛け算九九の総和を出力してください。

ただし、九九の結果が5の倍数の場合は九九の結果を二倍にしてから足し合わせてください。

## テストケース

input = none

```json
[
  {
    "input":,
    "output": [2450]
  }
]
```

## 模範回答

```python
n=0
for x in range(1,10):
    for y in range(1,10):
        if x*y % 5 ==0:
            n+=x*y*2
        else:
            n+=x*y
        
print(n)
```

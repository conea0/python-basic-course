## 問題

二重ループを使って、1から9の掛け算九九の総和を出力してください。

ただし、九九の結果が5の倍数になったら二重ループから抜けてください。

## テストケース

input = none

```json
[
  {
    "input":,
    "output": [400]
  }
]
```

## 模範回答

```python
n=0
for x in range(1,10):
    for y in range(1,10):
        if x*y % 5 ==0:
            break
        else:
            n+=x*y
        
print(n)
```

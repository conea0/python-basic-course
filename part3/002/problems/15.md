## 問題

16の約数の個数3つ以上なら、「素数ではない」を出力してください。
そうでないなら、「素数」を出力してください。

## テストケース

input = none

```json
[
  {
    "input":,
    "output": [True]
  }
]
```

## 模範回答

```python
x = 16
flg = False
if x % 2 == 0:
  flg = True
if x % 3 == 0:
  flg = True
if x % 5 == 0:
  flg = True
if x % 7 == 0:
  flg = True
if x % 11 == 0:
  flg = True
if x % 13 == 0:
  flg = True

if flg:
  print("素数ではない")
else:
  print("素数")
```

## 問題

二重ループを使って、1から9の掛け算九九の総和を出力してください。

ただし、九九の結果が6の倍数になったら二重ループから抜け、その時点での総和を二倍にしてください。

また、二倍した総和の下一桁が8であれば、Trueを出力してください。

そうでないなら、Falseを出力してください。

## テストケース

input = none

```json
[
  {
    "input":,
    "output": [True]
  }
]
```

## 模範回答

```python
n=0
for x in range(1,10):
    for y in range(1,10):
        if x*y % 6 ==0:
            break
        else:
            n+=x*y
        
if str(n*2)[-1] == "8":
    print(True)
else:
    print(False)
```

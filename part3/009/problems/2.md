## 問題

1からint(input())で入力された数までの整数を順番に表示するプログラムを作成してください。ただし、偶数の場合は表示せずに次の数に進んでください。また、入力した数まで表示が終わったらプログラムを終了してください。

## テストケース
input = none
```json
[
	{
		"input": ["2"],
		"output": ["1"]
  	},
]
```

## 模範回答
```python
b = int(input()) +1
for i in range(1, b):
    if i % 2 == 0:
        continue
    print(i)
```

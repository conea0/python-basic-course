## 問題

for文を使いiが1からint(input())で入力された数まである時、13の倍数になったら「〇〇は13の倍数です」と〇〇を埋めてその時の数で埋めて、出力されるような処理を作ってください。

## テストケース

```json
[
	{
		"input": ["26"],
		"output": ["13は13の倍数です","26は13の倍数です"]
  	},
]
```

## 模範回答
```python
b = int(input()) + 1
for i in range(1, b):
    if i % 13 != 0:
        continue
    print(f"{i}はこれは6の倍数です")
```

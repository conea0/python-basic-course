## 問題

list(map(int, input().split()))で入力されたリストの要素を一つずつ取り出して表示し、また最後にそのすべての合計を出力してください。

## テストケース

```json
[
	{
		"input": ["1 2 3 4"],
		"output": ["1","2","3","4","10"]
  	}
]
```

## 模範回答
```python
nums =list(map(int, input().split()))
for a in names:
	print(a)

total = sum(nums)
print(total)
```

## 問題

int(input())で入力した値に加えてその値が80以上であれば`優`、70以上であれば`良`、60以上であれば`可`、それ以外であれば`不可`と出力するようにしてください

例:72が入力された時は以下に表示する。<br>
"72 良"　
## テストケース
input = int
```json
[
	{
		"input": ["72"],
		"output": ["72 良"]
  	}
]
```

## 模範回答
```python
score =int(input())

if score >= 80:
	char = "優"
elif score >= 70:
	char = "良"
elif score >= 60:
	char = "可"
else:
	char = "不可"

print(score,char)    
```

## 問題

int(input())で入力した値を5で割った数の余りを`割った余りは(余り)です`と出力してください

## テストケース

```json
[
	{
		"input": ["7"],
		"output": ["割った余りは2です"]
  	}
]
```

## 模範回答
```python
num = int(input())

if num % 5 == 1:
	print("割った余りは1です")
elif num % 5 == 2:
	print("割った余りは2です")
elif num % 5 == 3:
	print("割った余りは3です")
elif num % 5 == 4:
	print("割った余りは4です")
else:
	print("割った余りは0です")
```

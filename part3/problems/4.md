## 問題

`while``for`を使って、1から3までをint(input())で入力した回数出力してください

## テストケース

```json
[
	{
		"input": ["5"],
		"output": ["1","2","3","1","2","3","1","2","3","1","2","3","1","2","3"]
	}
]
```

## 模範回答
```python
count = 0
b = int(input())
while True:
    for i in range(1,4):
        print(i)
	count += 1
    if count == b:
        break
```

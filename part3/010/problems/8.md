## 問題

`while`で、`count = int(0)`をはじめに作って、countをインクリデント`count += 1`していき、10回回ったら`break`してください。中身は`pass`だけとしてください

## 模範回答
```python
count = int(0)
while True:
    pass
    count += 1
    if count == 9:
        break
```

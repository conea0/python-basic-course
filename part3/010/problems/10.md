## 問題

`while`で、`count = int(0)`をはじめに作って、countをインクリデント`count += 1`していき`count = 15`になったら`break`してください。`3`で割り切れる場合のみ`count`を出力し、それ以外の場合は`pass`と書いてください

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": [3,6,9,12,15]
  	},
]
```

## 模範回答
```python
count = int(0)
while True:
    count += 1
    if count % 3 == 0:
        print(count)
    else:
        pass
    if count == 15:
        break
```

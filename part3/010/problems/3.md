## 問題

list(map(str,input().split()))で入力されたリストの各要素が2の倍数であれば"⚪︎は2の倍数"(⚪︎にはその時の要素がはいる)、そうでなければ`pass`を処理するプログラムを作って下さい.

## テストケース

```json
[
	{
		"input": ["3 2 1 5 4"],
		"output": ["2はの倍数","4は2の倍数"]
  	},
]
```

## 模範回答
```python
numbers = list(map(str, input().split()))

for num_str in numbers:
    num = int(num_str)
    if num % 2 == 0:
        print(f"{num_str}は2の倍数")
    else:
        pass 
```

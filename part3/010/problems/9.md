## 問題

`while`で、`count = int(0)`をはじめに作って、countをインクリデント`count += 1`していき`count`を出力してください、`count = 5`になったら`break`してください。出力の前後に`pass`と書いて処理を確認してください

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": [1,2,3,4,5]
  	},
]
```

## 模範回答
```python
count = int(0)
while True:
    count += 1
    pass
    print(count)
    pass
    if count == 5:
        break
```

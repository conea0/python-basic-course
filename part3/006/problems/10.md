```python
for i in 'リスト名':
	for j in 'リスト名':
```

## 問題

空リストl、リストl1=[1, 2, 3]、l2=[101, 102, 103]を作成し、for文を使ってi*jの値を求め、リストlに追加してください。なお、追加したら結果を出力してください。

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": [[100, 101, 102, 200, 202, 204, 300, 303, 306]]
  	}
]
```

## 模範回答
```python
l = []
l1 = [1, 2, 3]
l2 =[100, 101, 102]
a = 0
for i in l1:
  for j in l2:
    a =i*j
    l.append(a)
print(l)
```

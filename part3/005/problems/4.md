## 問題

二重ループを使って、1から9の掛け算九九の総和を出力してください。

ただし、九九の結果が三の倍数の場合は足し合わせないでください。

## テストケース

input = none

```json
[
  {
    "input":,
    "output": [729]
  }
]
```

## 模範回答

```python
n=0
for x in range(1,10):
    for y in range(1,10):
        if x*y % 3 ==0:
            continue
        n+=x*y
        
print(n)
```

## 問題

`while`を使って、list(map(str, input().split()))で入力されたリストのそれぞれ要素を`私の名前は〇〇です`の〇〇に当てはめて1回目はそのまま、２回目はその名前の後ろに１を追加して出力してください

※入力が"taro goro"だとすると、"私の名前はgoroです","私の名前はtaroです","私の名前はgoro1です","私の名前はtaro1です"と出力するようにする。
## テストケース

```json
[
	{
		"input": ["goro taro jiro nanashi"],
		"output": ["私の名前はgoroです","私の名前はtaroです","私の名前はjiroです","私の名前はnanashiです"]
	}
]
```

## 模範回答
```python
names = list(map(str, input().split()))
count = 0

while True:
    for name in names:
        print(f"私の名前は {name} {'1'*count} です")
    count += 1

    if count == 2:
        break

```

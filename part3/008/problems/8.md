## 問題

`while True`を使ってlist(map(int, input().split()))で入力されたリストの中身をすべて出力してください

## テストケース

```json
[
	{
		"input": ["3 5 2 0"],
		"output": ["3","5","2","0"]
	}
]
```

## 模範回答
```python
l = list(map(int, input().split()))
i = 0
while True:
    if len(l) == i:
        break
    print(l[i])
    i += 1
```

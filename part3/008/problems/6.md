## 問題

`while``for`を使って、1から3までをint(input())で入力した回数出力してください。<br>
出力する時は、すべて元の数を２を割ってください。


## テストケース

```json
[
	{
		"input": ["3"],
		"output": ["2","4","6","2","4","6","2","4","6"]
	}
]
```

## 模範回答
```python
count = 0
b = int(input())
while True:
  for i in range(1,4):
    print(i / 2)
  count += 1
  if count == b:
    break
```

## 問題

x+y=zのとき"xとyの和はzと等しい"と出力されるようなif文を作って、出力してください。

```x=int(input())```,```y=int(input())```,```z=int(input())```として入力しなさい。（xは25、yは25 ,z=200 としよう)
## テストケース

```json
[
	{
		"input": ["25","25","200"],
		"output": ["xとyの和はzと等しい"]
  	},
]
```

## 模範回答
```python
x = int(input())
y = int(input())
z = int(input())
if x + y == z:
    print("xとyの和はzと等しい")
```

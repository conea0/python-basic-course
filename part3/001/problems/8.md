## 問題

x+y==5のとき"xとyの和は5である"と出力されるようなif文を作って、出力してください。

```x=int(input())```,```y=int(input())```として入力しなさい。（xは2、yは3としよう)
## テストケース

```json
[
	{
		"input": ["2","3"],
		"output": ["xとyの和は5である"]
  	},
]
```

## 模範回答
```python
x = int(input())
y = int(input())
if x + y == 5:
    print("xとyの和は5である")
```

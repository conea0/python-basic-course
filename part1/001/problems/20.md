## 問題

`lst`に、インデックスの0~2にそれぞれ`lst_0 = ["ぬ","す","ろ"]``lst_1 = [10,80,30,70]``lst_2 = ["r","n","a","y","g"]`を代入した後、そこから`sum`を使ってを数字のリストの中身の合計出力してください。

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["や"]
	}
]
```

## 模範回答
```python
lst_0 = ["ぬ","す","ろ"]
lst_1 = [10,80,30,70]
lst_2 = ["r","n","a","y","g"]

lst = [lst_0,lst_1,lst_2]
print(sum(lst[1]))
```

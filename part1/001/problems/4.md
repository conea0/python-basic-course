## 問題

`while``for`を使って、初めに`count = int(1)`を作り、インクリデントを使って5回、1~3を出力してください

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": [1,2,3,1,2,3,1,2,3,1,2,3,1,2,3]
	}
]
```

## 模範回答
```python
count = int(1)
while True:
    for i in range(3):
        print(i)
	count += 1
    if count == 5:
        break
```

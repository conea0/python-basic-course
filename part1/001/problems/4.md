## 問題

`while``get_input``for`を使って、`num_1`に代入した回数1~3を出力してください
※`count = 0`という変数を使ってみるといいでしょう

## テストケース
input = int
```json
[
	{
		"input": [5],
		"output": [1 2 3 1 2 3 1 2 3 1 2 3 1 2 3]
	}
]
```

## 模範回答
```python
num_1 = get_input()
count = 0
while True:
    for i in range(3):
        print(i)
	count += 1
    if count == num_1:
        break
```

## 問題

リスト s=["baseball", "softball", "volleyball", "handball", "golf", "tennis", "football"] の0番目から5番目の要素を2個刻みに取り出して出力してください。

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["['baseball', 'volleyball', 'golf']"]
  	},
]
```

## 模範回答
```python
s = ["baseball", "softball", "volleyball", "handball", "golf", "tennis", "football"]
print(s[0:6:2])
```

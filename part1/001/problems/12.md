## 問題

`setdefault` を使用して`nombers_sd = {3 : "nomber_1",4 : "nomber_2",1 : "nomber_3"}`の中に`2`の第一引数だけ追加した後、`get`を使用して`2`の第二引数を出力してください

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["None"]
  	},
]
```

## 模範回答
```python
nombers_sd = {2 : "nomber_1",3 : "nomber_2",1 : "nomber_3"}
nombers_sd.setdefault(2)
print(nombers_sd.get(2))
```

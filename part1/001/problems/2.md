## 問題

`get` を使用して`nombers_g = {2 : "nomber_1",3 : "nomber_2",1 : "nomber_3"}`から`"nomber_3"`を出力してください

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["nomber_3"]
  	},
]
```

## 模範回答
```python
nombers_g = {2 : "nomber_1",3 : "nomber_2",1 : "nomber_3"}
print(nombers_g.get(1))
```

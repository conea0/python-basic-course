## 問題

`while``get_input`を使って、何回も`私の名前は(get_input)です`を出力してください。また、`end`と入力することで`bleak`して`終了します`と出力してください

## テストケース
input = str
```json
[
	{
		"input": ["goro"],
		"output": ["私の名前はgoroです"]
	}
]
```

## 模範回答
```python
while True:
	name = get_input()
	if name == "end":
		print("終了します")
		break
	print("私の名前は" + name + "です")
```

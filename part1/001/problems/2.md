## 問題

`get` を使用して`numbers_g = {2 : "uomber_1",3 : "number_2",1 : "number_3"}`から`"number_3"`を出力してください

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["number_3"]
  	},
]
```

## 模範回答
```python
nombers_g = {2 : "number_1",3 : "number_2",1 : "number_3"}
print(nombers_g.get(1))
```

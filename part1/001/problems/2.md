## 問題

`get_input`を使用して、`イヌ`であれば`動物`、`マグロ`であれば`さかな`、`にわとり`であれば`とり`、それ以外であれば`判別できない`と出力してください

## テストケース
input = str
```json
[
	{
		"input": ["ネコ"],
		"output": ["判別できない"]
  	}
]
```

## 模範回答
```python
name = get_input()

if name == "イヌ":
	print("動物")
elif name == "マグロ":
	print("さかな")
elif name == "にわとり":
	print("とり")
else :
	print("判別できない")
```

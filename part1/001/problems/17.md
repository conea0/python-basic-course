## 17

num1に代入した数字を一辺(m)として、平方メートル(m＾2)に直し、それが何倍になれば1キロ平方メートル(1km＾2)になるのかを出力してください
※円周率は3.14としてください

## テストケース
input = float
```json
[
	{
		"input": ["1"],
		"output": ["1000000.0"]
  	},
	{
		"input": ["125"],
		"output": ["64.0"]
	}
]
```

## 模範回答
```python
num1 = get_input()
print((1000 ** 2) / (num1 ** 2))
```

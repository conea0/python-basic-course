## 15

num1に代入した数字を底面の正方形の一辺、num2に代入した数字を高さとして四角柱の体積を出力してください
※四角柱は[底面 * 高さ]で求めることができる。さらに、底面は正方形なので、一辺の長さがわかれば求めることができる
## テストケース
input = float
```json
[
	{
		"input": ["2"],["3"]
		"output": ["12.0"]
	}
]
```

## 模範回答
```python
num1 = get_input()
num2 = get_input()
print(num1 * num1 * num2)

print((num1 ** 2) * num2)
```

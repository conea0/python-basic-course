## 問題

`lst`に、インデックスの0~2にそれぞれ`lst_0 = ["l","e","h","w"]``lst_1 = ["ろ","お","な"]``lst_2 = [10,70,60]`を代入した後、そこから`len`を使って`4`を出力してください。

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": [4]
	}
]
```

## 模範回答
```python
lst_0 = ["l","e","h","w"]
lst_1 = ["ろ","お","な"]
lst_2 = [10,70,60]

lst = [lst_0,lst_1,lst_2]
print(min(lst[0]))
```

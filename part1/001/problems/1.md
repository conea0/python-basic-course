## 問題

`get` を使用して`words_g = {"word_1" : "りんご","word_2" : "すいか","word_3" : "みかん"}`から`すいか`を出力してください

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["すいか"]
  	},
]
```

## 模範回答
```python
words_g = {"word_1" : "りんご","word_2" : "すいか","word_3" : "みかん"}
print(words_g.get("word_2"))
```

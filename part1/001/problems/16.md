## 16

num1に代入した数字を底面の円の直径、num2に代入した数字を高さとして四角柱の体積を出力してください
※円周率は3.14としてください

## テストケース
input = float
```json
[
	{
		"input": ["2"],["1"]
		"output": ["3.14"]
  	},
	{
		"input": ["4"],["6"]
		"output": ["75.36"]
	}
]
```

## 模範回答
```python
num1 = get_input()
num2 = get_input()
print(((num1/2) * (num1/2) * 3.14) * num2)

print(((num1/2) ** 2) * 3.14 * num2)
```

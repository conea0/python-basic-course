## 問題

`setdefault` を使用して`nombers_sd = {3 : "nomber_1",1 : "nomber_2",2 : "nomber_3"}`の中に`4`の第一引数だけ追加して、第二引数には何が入っているか出力してください

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["nomber_3"]
  	},
]
```

## 模範回答
```python
nombers_sd = {2 : "nomber_1",3 : "nomber_2",1 : "nomber_3"}
nombers_sd.setdefalt(4)
print(nombers_sd)
```

## 問題

`while``get_input`を使って、`num_1``num_2`入力した二つの数字を足した数字を出力してください。また、足した数字が100以上であれば`break`して下さい

## テストケース
input = int
```json
[
	{
		"input": [32,64],
		"output": [98]
	}
]
```

## 模範回答
```python
while True:
	num_1 = get_input()
    num_2 = get_input()
	if num_1 + num_2 >= 100:
		break
	print(num_1 + num_2)
```

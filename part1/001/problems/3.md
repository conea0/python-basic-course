## 問題

`setdefault` を使用して`numbers_sd = {3 : "number_1",1 : "number_2",2 : "number_3"}`の中に`4`の第一引数だけ追加して、第二引数には何が入っているか出力してください

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["nomber_3"]
  	},
]
```

## 模範回答
```python
numbers_sd = {2 : "nomber_1",3 : "nomber_2",1 : "nomber_3"}
numbers_sd.setdefault(4)
print(numbers_sd)
```

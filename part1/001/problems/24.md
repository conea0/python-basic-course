## 問題

copyメソッドを使って、リストl=['2023','September', '9', 'Saturday', 'rain']をコピーして出力してください。

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["['2023','September', '9', 'Saturday', 'rain']"]
  	},
]
```

## 模範回答
```python
l = ['2023','September', '9', 'Saturday', 'rain']
l2 = l.copy()
print(l2)
```

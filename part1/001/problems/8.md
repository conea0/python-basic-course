## 問題

`get_input`を使用して、`sales`が20未満であれば`年は20未満です`、40未満であれば`年は40未満、20以上です`、60未満であれば`年は60未満、40以上です`、それ以外であれば`年は60以上です`と出力するようにしてください

## テストケース
input = int
```json
[
	{
		"input": [46],
		"output": ["年は60未満、40以上です"]
  	}
]
```

## 模範回答
```python
sales = get_input()

if sales <= 20:
	print("年は20未満です")
elif sales <= 40:
	print("年は40未満、20以上です")
elif sales <= 60:
	print("年は60未満、40以上です")
else:
	print("年は60以上です")
```

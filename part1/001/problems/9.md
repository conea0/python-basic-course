## 問題

空リストl、タプルt=('Apple', 'Orange', 'Banana')を作成し、for文を使ってタプルの中身を空リストに移し（空リストに追加し）結果を出力してください。

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["['Apple', 'Orange', 'Banana']"]
  	}
]
```

## 模範回答
```python
t = ('Apple', 'Orange', 'Banana')
l = []
for animal in t:
  l.append(animal)
print(l)
```

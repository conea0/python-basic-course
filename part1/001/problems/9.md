## 問題

リストl=['tulip', 'jasmine', 'carnation', 'lavender', 'cosmos']の2番目にflowerを追加して出力してください。

## テストケース
input = none
```json
[
	{
		"input": [],
		"output": ["['tulip', 'jasmine', 'flower', 'carnation', 'lavender', 'cosmos']"]
  	}
]
```

## 模範回答
```python
l = ['tulip', 'jasmine', 'carnation', 'lavender', 'cosmos']
l.insert(2, 'flower')
print(l)
```

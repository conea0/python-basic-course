## 問題

`while``get_input`を使って、`num_1`に代入した数字num_1のあまりが7`3`なら`余りは3`、それ以外であればnum_1を1足して`(余りの数字) : 余りは3ではない`と出力してください。また、num_1に代入した数字が`6以下`であれば`出力なし`と出力してください

## テストケース
input = int
```json
[
	{
		"input": [],
		"output": [9 7 5 3 1]
	}
]
```

## 模範回答
```python
num_1 = get_input()
while True:
    if num_1 <= 6:
        print("出力なし")
        break
    if num_1 % 7 == 3:
        print("余りは3")
        break
    print(num_1 % 7,": 余りは3ではない")
    num_1 += 1
```
